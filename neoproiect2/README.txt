Netflix Titles CSV Viewer
This PHP script allows you to view and search through a CSV file containing Netflix titles. The script provides a simple web interface to filter and paginate the data.

Features

Search for titles by keyword
Paginate results with limit and offset controls
Displays search results in a table with 12 columns
Displays the number of search results and page load time

Place the netflix_titles.csv file in the same directory as this script.
Open the script in a web browser to access the interface.
Use the search, limit, and offset fields to filter and paginate the data.
Click the "Apply" button to apply the filters and display the results.
Note

This script assumes that the netflix_titles.csv file has 12 columns. If your CSV file has a different structure, you may need to modify the script accordingly.
